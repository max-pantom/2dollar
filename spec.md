# 2DollarDomain Spec

## 1. Product Summary

2DollarDomain is a fun domain discovery site that helps people find cheap available domains for ideas they want to build.

It is not only for joke ideas. It is for serious ideas, tiny SaaS ideas, creator brands, newsletters, experiments, African startups, side projects, and random internet thoughts that deserve a link.

The core promise:

> Search an idea. Find cheap available domains. See the real renewal price before you buy.

## 2. Product Positioning

2DollarDomain is a domain vending machine for internet ideas.

It should feel playful, fast, and slightly chaotic, but still trustworthy.

The site should not behave like a normal domain registrar. It should feel like a place where ideas become real quickly.

## 3. Core User

### Primary users

1. Indie hackers
2. Designers
3. Developers
4. Content creators
5. Students
6. African founders
7. Side project builders
8. People on X who get random ideas and want to claim a domain fast

### User mindset

The user is not always searching for a domain because they already have a company.

Sometimes they are thinking:

```txt
I just got an idea.
I do not know if I will build it.
But if the domain is cheap, I might buy it.
```

## 10. Roadmap

### 10. Almost bought wall

A public wall of domains people almost bought.

This should feel social, voyeuristic, and lightweight. People love seeing what others nearly claimed.

#### Almost bought today

- `rentafounder.xyz`
- `buttonfarm.site`
- `auntieapi.fun`
- `sponsorcrate.xyz`

#### Feature notes

- Show the promo price next to each domain.
- Add a claim post generator for domains someone actually buys.

If someone buys a domain, let them generate:

```txt
I bought fanledger.site for $1.98.

No idea if I’ll build it.

```
