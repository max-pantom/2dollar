# 2DollarDomain

A playful static MVP for finding cheap available domains around $2.

The first milestone is intentionally frontend-first:

- Search an idea, name, or exact domain.
- Generate sample domain cards from template logic.
- Show first year price, renewal price, registrar, and warning badges.
- Link out to a registrar checkout/search page.
- Share a result to X.
- Browse daily drop and random mode pages.

Real registrar integrations should replace the static generator later through
the API routes in `app/api`.

## Development

Run the app:

```bash
pnpm dev
```
